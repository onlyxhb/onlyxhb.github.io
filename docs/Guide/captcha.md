# 获取动态码

动态码用于登录编程导航、永久解锁主页模板等。

**1. 扫码关注微信公众号『 编程导航 』**

![编程导航二维码](https://636f-codenav-8grj8px727565176-1256524210.tcb.qcloud.la/assets/qrcode.jpg)

**2. 在公众号内点击 "一键登录" 或回复 "登录" 获取**

如果点了后没反应，请再点一次 😢
