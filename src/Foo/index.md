# 相关链接

## 介绍

文档工具可以帮助您开发库和编写文档。

### 特征

- 📦 开箱即用，专注于组件开发和文档
- 📋 丰富的 Markdown 扩展，不仅仅是渲染组件演示
- 🏷 根据 TypeScript 类型定义自动生成组件 API
- 🎨 轻松自定义主题，创建自己的 Markdown 组件
- 📱 支持移动组件库开发，内置移动高清渲染方案
- 📡 一站式命令将组件资产数字化并将其与下游生产力工具连接起来
